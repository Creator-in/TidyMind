# TidyMind
AI-powered mood planner that turns your emotions into action


Steps for installation:

1. Download the repo from the branch 'sample'

2. Navigate to Project

    ```
    cd TidyMind
    ```
3. Install Packages

    ```
    npm install
    ```

4. Run the App

    ```
    npm run dev
    ```

The website is available at http://localhost:5173
